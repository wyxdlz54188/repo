# NewTerm

**版本 1.0**

---

## 简介

NewTerm 是一个为 iOS 6 设计的终端模拟器，基于经典的 MobileTerminal 项目二次开发。它提供了完整的 VT100 终端模拟功能，让你在 iOS 设备上享受原生的终端体验。

---

## 技术信息

| 项目 | 详情 |
|------|------|
| 终端引擎 | VT100 |
| 渲染引擎 | CoreText |
| 最低系统 | iOS 4.0 |
| 目标系统 | iOS 6.0 |
| 支持架构 | ARMv7 / ARMv7s |

---

## 功能特性

- 完整的 VT100 终端模拟
- 支持 256 色显示
- CoreText 高性能文本渲染
- 多窗口终端管理
- 手势操作（九宫格触控）
- 自定义字体和配色
- URL Scheme 动态配置

---

## 手势操作指南

| 区域 | 点击 | 长按 |
|------|------|------|
| 左上 | Insert | - |
| 上方 | Page Up | 方向键上（重复） |
| 右上 | Delete | 关闭窗口 |
| 左侧 | Home | 方向键左（重复） |
| 中央 | Ctrl Lock | 显示菜单 |
| 右侧 | End | 方向键右（重复） |
| 左下 | Esc | - |
| 下方 | Page Down | 方向键下（重复） |
| 右下 | Tab | 窗口列表 |

- 左右滑动：切换终端窗口
- 双指长按：显示/隐藏键盘
